# Learnings

_Distilled principles. Synthetic example data._

### 2026-04-21 — capture is cheap, distillation is the work

Every agent build I've done failed at the distillation step, not the capture step. Capture is one webhook. Distillation needs a stable taxonomy AND a stable voice. Get the taxonomy right first.

### 2026-04-12 — markdown ages better than schemas

Three different tools have lost me data through schema migrations. Markdown is forgiving. Even a malformed brain/raw.md is greppable. Schemas need migrations; text doesn't.

### 2026-04-02 — "self-improving" beats "feature-complete" for personal tools

Personal tools that improve themselves through use compound. Personal tools that ship feature-complete and don't iterate get abandoned within 90 days. Optimize for "boring loop ran today."
