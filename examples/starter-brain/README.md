# starter brain

Minimal layout. Copy this directory, run `bash <nanobrain>/install.sh <copy>`, and start ingesting.

```
brain/
  self.md       # who you are, voice, style
  goals.md      # what you're working toward
  projects.md   # active threads
  people.md     # contacts (index)
  decisions.md  # decisions log (append-only)
  learnings.md  # lessons (append-only)
  raw.md        # full mirror of distilled signal
  interactions.md  # append-only log (firehose, not agent-readable)
_contexts.yaml  # solo personal default; add `work: {}` if you have one
```
